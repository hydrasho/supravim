## Installation:


 Version 4.5.5: 
```bash
curl https://gitlab.com/hydrasho/SupraVim/-/raw/master/installer.sh | sh
```

 Version 5.0 (dev-stable):
```bash
curl https://gitlab.com/hydrasho/SupraVim/-/raw/suprapack/install.sh | sh
```



#### Mise à jour:
```bash
    supravim update
```

#### Désinstallation

```bash
    supravim uninstall
```
